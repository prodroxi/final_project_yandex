# Тесты для сервиса аренды самокатов

## Описание
Автоматизированные тесты для проверки API сервиса аренды самокатов.

## Тест-кейс
1. Создание заказа через API
2. Получение созданного заказа по номеру трека
3. Проверка, что код ответа равен 200

## Требования
- Python 3.7+
- Установленные зависимости -pip install requests
